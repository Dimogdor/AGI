CE QU'ON PEUT VOUS DIRE

Deux camps. Une seule issue : détruire la base d'en face avant qu'elle ne détruise la vôtre. Recrutez, bâtissez, tenez l'eau — tout le reste s'apprend en regardant le monde changer. Et il va changer.

L'ESSENTIEL

Touchez les socles « + » pour construire. Touchez vos bâtiments et votre base pour les entretenir — ils vous le rendront. Postez des troupes près des lacs pour les revendiquer. Frappez l'ennemi pour gagner ✦ et évoluer. Jusqu'où ? À vous de le découvrir.

SÉLECTION — PC

Clic sur une unité = son groupe ; re-clic = juste elle ; clic au sol = désélection. Lasso : maintenez Maj + glissez, ou bouton ⬚ Lasso. Molette = caméra · +/− = zoom · touches modifiables dans Paramètres.

SÉLECTION — SMARTPHONE (paysage conseillé)

Toucher une unité = son groupe ; re-toucher = juste elle ; toucher le sol = désélection. ⬚ Lasso puis glissez le doigt. Glisser sur le terrain = caméra. Pincez pour zoomer.
